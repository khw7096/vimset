# vim setting
* 개인 Vim 셋팅입니다.

##### install
```
cd ~ && git clone https://github.com/khw7096/vimset.git && cd vimset && sh install.sh && rm -rf ~/vimset
```

#### Hotkey
- F8 : run python
- F9 : run golang
